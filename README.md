### 项目地址

```bash
git clone https://github.com/moedong/nuxt-iview.git
```
### 安装
```bash
npm install
```

### 开发
```bash
npm run dev

// GO to http://127.0.0.1:3000
```

### 测试环境
```bash
// 需要pm2，如果没有安装的先安装  npm install -g pm2
npm run publish:test
```

### 生产环境
```bash
// 需要pm2，如果没有安装的先安装  npm install -g pm2
npm run publish:prod
```
